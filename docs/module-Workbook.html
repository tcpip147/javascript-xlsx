CellStyle createCellStyle(Object styles)

셀 스타일을 생성한다.
Example
var workbook = JavascriptXlsx.createWorkbook();
var sheet = workbook.createSheet("Sheet1");
var row = sheet.createRow(0);
var cell = row.createCell(0);
var style = workbook.createCellStyle({
    font: {
        size: 20
    },
    border: {
        left: {
            style: "thin",
            color: "FF0000"
        }
    }
});
cell.setCellValue("Hello");
cell.setCellStyle(style);

Number getActiveSheetIndex()

현재 활성화 된 시트의 Index를 반환한다.
Example
var workbook = JavascriptXlsx.createWorkbook();
workbook.createSheet("Sheet1");
workbook.createSheet("Sheet2");
workbook.setActiveSheet(1);
var index = workbook.getActiveSheetIndex();
console.log(index); // 1

Sheet getSheet(String name)

이름으로 시트를 가져온다.
Example
var workbook = JavascriptXlsx.createWorkbook();
var sheet1 = workbook.createSheet("Sheet1");
var sheet2 = workbook.createSheet("Sheet2");
var sheet = workbook.getSheet("Sheet2");
console.log(sheet2 === sheet); // true
